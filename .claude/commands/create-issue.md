---
description: GitHub 이슈 생성
---

사용자가 요청한 내용을 바탕으로 GitHub 이슈를 생성합니다.

## 입력
$ARGUMENTS

## 작업 절차

1. 입력된 내용을 분석하여 적절한 이슈 제목과 본문을 작성합니다.
2. 이슈 본문은 다음 형식을 따릅니다:
   - 개요: 이슈에 대한 간단한 설명
   - 상세 내용: 구체적인 요구사항이나 문제 설명
   - (필요시) 체크리스트: 해야 할 작업 목록

3. `gh issue create` 명령어를 사용하여 이슈를 생성합니다.

## 이슈 생성 명령어 예시
```bash
gh issue create --title "이슈 제목" --body "이슈 본문"
```

## 주의사항
- 이슈 제목은 명확하고 간결하게 작성합니다.
- 본문은 마크다운 형식으로 작성합니다.
- 생성 후 이슈 URL을 사용자에게 알려줍니다.
